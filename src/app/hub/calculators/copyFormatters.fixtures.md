# Calculator Quote Copy Fixtures

These examples document the stable customer-facing quote copy contracts for EU Standard and US Client calculators.
Do not rewrite the layout unless the requested change explicitly targets quote copy.

## EU Standard

### Custom label, print and embroidery

```text
Test dEsg:

JH001 AWDis College hoodie (1 Col Front, Embroidery 1, Embroidery 2)
Digitizing fee = €50 (incl vat)
50 x €17.58 (excl vat) ea = €1116.33
```

### Blank label fallback, print only

```text
Item #1:

JH001 AWDis College hoodie (1 Col Front)
50 x €16.31 (excl vat) ea = €1035.69
```
